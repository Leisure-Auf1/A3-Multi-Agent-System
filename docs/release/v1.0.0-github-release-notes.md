## Highlights

### 🤖 7-Agent Workflow
ProfileAgent → PlannerAgent → ContentGenerator → ResourceAgent → ReviewGate → ReflectionAgent → Memory

Describe what you want to learn in natural language. The agent team profiles your cognitive style, builds a personalized learning path, generates materials, and evaluates quality — all automated through an EventBus pipeline.

### 🔒 Production Security
- JWT authentication with PBKDF2-SHA256 password hashing
- Role-based permissions (free / pro / teacher / admin)
- Token budget enforcement per user
- Audit logging to filesystem (JSONL per user)
- All endpoints require authentication

### 💾 Persistent Memory
- Full user lifecycle: Register → Login → Run → Logout → Login → Restore
- SQLite for users, sessions, profiles, learning records
- Filesystem workspace for artifacts (per-user isolated)
- Veritas StudentMemory for long-term learning state

### 🖥️ Cross-Platform Deployment
| Platform | Method |

### 📊 2661 Tests
- Full API coverage (FastAPI TestClient)
- Security: auth, permissions, token budget

- UI: API client, data providers, theme

## Installation

### Source (any platform)
```bash
git clone https://github.com/Leisure-Auf1/A3-Multi-Agent-System.git
cd A3-Multi-Agent-System
pip install -r requirements.txt
streamlit run web/app.py
```

### Docker
```bash
docker pull ghcr.io/leisureauf1/a3-multi-agent-system:v1.0.0
docker run -p 8501:8501 -p 8000:8000 ghcr.io/leisureauf1/a3-multi-agent-system:v1.0.0
```

### Windows Desktop
Download `A3-Agent-v1.0.0-win64.zip`, extract, double-click `A3-Agent.exe`.

### Linux Desktop
```bash
tar xzf A3-Agent-v1.0.0-linux-x64.tar.gz
cd A3-Agent-v1.0.0-linux-x64 && ./A3-Agent
```
