# Python for AI Engineers

> Course Code: PY-AI-201
> Level: Beginner → Intermediate
> Estimated Duration: 60 hours (20 chapters × 3 hours)
> Prerequisites: Basic computer literacy, no prior programming required

---

## Course Description

This course provides a structured path from zero programming experience to building AI applications with Python. Students start with fundamental programming concepts and progress through data science libraries (NumPy, Pandas), machine learning basics, neural networks, and finally Large Language Models and multi-agent systems.

The curriculum is designed for aspiring AI engineers who need both strong Python fundamentals and practical AI/ML skills. Every chapter includes hands-on exercises and real-world examples.

---

## Learning Objectives

After completing this course, students will be able to:

1. **Write** clean, idiomatic Python code for data processing and automation
2. **Manipulate** data efficiently using NumPy arrays and Pandas DataFrames
3. **Understand** the machine learning workflow from data preparation to model evaluation
4. **Build** and train simple neural networks for classification and regression
5. **Explain** how Large Language Models work and how to prompt them effectively
6. **Construct** Retrieval-Augmented Generation (RAG) systems
7. **Design** multi-agent AI systems using modern agent frameworks
8. **Deliver** a complete AI engineering capstone project
